GETTING STARTED
===============

INSTANT STARTUP (No Installation):
1. Locate project folder: c:\Users\hp\Desktop\project 7
2. Double-click index.html
3. View in your browser immediately!

USING LOCAL SERVER (Recommended):

Option A: Python (Most Common)
   1. Open terminal/command prompt
   2. Navigate to project folder
   3. Run: python -m http.server 8000
   4. Open: http://localhost:8000

Option B: Python 2
   1. Run: python -m SimpleHTTPServer 8000
   2. Open: http://localhost:8000

Option C: Node.js
   1. Install globally: npm install -g http-server
   2. Run: http-server
   3. Open: http://localhost:8080

Option D: VSCode Live Server
   1. Install "Live Server" extension
   2. Right-click index.html
   3. Select "Open with Live Server"


CUSTOMIZATION GUIDE
===================

Change Colors:
   1. Open css/style.css
   2. Find :root { } section (top of file)
   3. Modify --color-primary, --color-secondary
   4. Save and refresh browser

Update Company Name:
   1. Open index.html
   2. Find "Digital Solutions" text
   3. Replace with your company name
   4. Update in navbar, logo, footer

Replace Logo:
   1. Save your logo image to assets/images/
   2. OR edit the SVG code in the navbar section
   3. Replace the placeholder SVG

Update Content:
   1. Open index.html
   2. Find section you want to edit
   3. Replace text with your content
   4. Save and refresh

Add Your Images:
   1. Save images to assets/images/
   2. Replace placeholder SVGs with img tags
   3. Use appropriate alt text

Update Contact Information:
   1. Find contact section in index.html
   2. Update email, phone, address
   3. Update form action if sending to backend

Add Social Media Links:
   1. Find footer social links section
   2. Update href attributes
   3. Add your social media URLs
