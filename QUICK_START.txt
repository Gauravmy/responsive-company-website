INSTANT STARTUP (No Installation Required!)
============================================

1. OPEN THE PROJECT
   - Navigate to the project folder
   - Look for: index.html
   - Double-click index.html to open in your default browser
   - OR right-click > Open with > Your favorite browser

2. USING A LOCAL SERVER (Recommended)

   Option A: Python (Most Common)
   - Open terminal/command prompt in project folder
   - Type: python -m http.server 8000
   - Open: http://localhost:8000 in browser
   
   Option B: Using Python 2
   - Type: python -m SimpleHTTPServer 8000
   
   Option C: Using Node.js
   - Install globally: npm install -g http-server
   - Type: http-server
   - Open: http://localhost:8080 in browser
   
   Option D: VSCode Live Server Extension
   - Install extension: "Live Server" by Ritwick Dey
   - Right-click index.html
   - Select: "Open with Live Server"

CUSTOMIZATION TIPS
